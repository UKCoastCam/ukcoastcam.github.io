Help & How to Use

Finding Webcams on the Map

The main page features an interactive map showing the locations of coastal, harbour and maritime webcams. Each camera is marked with a map pin.

Searching for Places

Click the magnifying glass on the map to search for places. Start typing a city, beach, harbour, or landmark, then choose from the suggestions.

Webcam Clusters

When zoomed out, webcams in the same area appear as clusters. Zoom in to separate them into individual pins.

Viewing a Webcam

Click any map pin to view the webcam and open its dedicated page in a new tab.

Submitting or Reporting Webcams

Use the Submit or Report buttons on the main page to help improve the directory.
